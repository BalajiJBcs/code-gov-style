# Focusable Card
You can make a card react when a user hovers over it by adding a `focusable` class
<div style="max-width: 400px">
  <div class="card focusable">
    <p>Hi. I'm a <b>focusable</b> card.  Hover over me!</p>
  </div>
</div>
