# Indentation
You can indent a div by simply adding class="indented".
<div><p>I'm <b>not</b> indented</p></div>
<div class="indented"><p>Hey, I'm an <b>indented</b> block</p></div>
<div class="indented"><p>I'm <b>indented</b>, too!</p></div>
